SOBRE O EVENTO

A Python Brasil é a maior conferência da linguagem de programação Python da América Latina, um evento feito pela comunidade para a Comunidade.

Temos orgulho de ter um ambiente seguro, amistoso, inclusivo e acessível, onde todas as pessoas são bem-vindas e possam se sentir à vontade, aproveitando o melhor que o evento tem a oferecer.

O LOCAL DO EVENTO

As palestras acontecerão no Centro de Convenções Bolsa de Valores, na Praça XV, um local central no Rio de Janeiro, em um auditório com capacidade para até 1000 pessoas.

Python Brasil é uma conferência sem fins lucrativos dirigida por voluntários para promover a linguagem de programação Python de código aberto. A conferência é apoiado pela Associação Python Brasil (APyB).
